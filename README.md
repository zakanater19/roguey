an attempt to port roguetown ss13 into ss14, AGPL 3.0 on some of the assets used from the archive found here https://github.com/SS13-Special-Codebases-Archive/RogueTown


This is all very WIP and not yet a real project, ill update here with videos and more text when it's near ready.



New keybinds;
hold space to sprint, press R to throw

New console commands;
day - Changes time to day, night - Changes time to night - timespeed 1-20 Changes day/night cycle and time speed

regenerate - server already populated debug bog tiles by default but you can reload them or load them with this command
ungenerate - deletes all the trees and tiles and stuff from debug bog tiles, use this when mapping, you don't need to regenerate when saving the server autoloads regenerate on start.
